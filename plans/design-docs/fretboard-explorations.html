Fretboard Design Explorations

5 directions for the guitar fretboard — quiz highlighting + stats heatmap states

1. Transparent Text

No circles at all. Note names float directly on the strings. Stats are conveyed through text color intensity (terracotta → sage scale). Active quiz note uses a warm gold. Clean and airy — lets the fretboard structure breathe.

2. Dot Grid

Even grid spacing (not realistic fret distances). Small filled dots for each position — minimal footprint, great for data-dense views. Hover reveals the note name in a tooltip. Stats shown by dot color. Compact and information-rich.

3. Vertical Dark

Rotated 90° — strings run horizontally, frets descend vertically, matching the perspective of looking down at your guitar. Dark background with warm string tones. Dots are subtle; note names appear on hover. Stats glow softly against the dark surface.

4. Cell Grid

GitHub-contributions-style grid. Each fretboard position is a rounded cell with the note name inside. Even spacing, no fret lines or strings — pure data visualization. Cells color-fill for stats. Great for the analytics dashboard use case.

5. Warm Wood / Glass

Premium feel: dark wood-toned background with warm string colors. Notes live in subtle frosted circles that glow on hover. Stats use translucent colored fills that feel layered on the surface. Realistic fret spacing preserved. The most "physical instrument" of the five.
